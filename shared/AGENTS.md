# Agentes — Projeto Conceitto

Este projeto segue os padrões instalados pelo [conceittodev-tools](https://github.com/devgabrielmichel/teste-tools).

## Estrutura

- `frontend/` — interface (React)
- `backend/` — APIs (Node)
- `docs/` — documentação técnica (Markdown)
- `tests/` — testes automatizados

## Cursor

- Rules em `.cursor/rules/` — padrões Conceitto e stack
- Skills em `.cursor/skills/` — fluxos específicos (descritivo, docs, etc.)
