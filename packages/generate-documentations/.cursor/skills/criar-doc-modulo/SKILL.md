# Criar documentação de módulo

1. Identifique o módulo/tela (nome claro, kebab-case no arquivo).
2. Copie a estrutura de `docs/modulos/TEMPLATE.md`.
3. Salve em `docs/modulos/<nome>.md` ou `docs/<area>/<nome>.md`.
4. Linguagem para desenvolvedores e gestores técnicos — não confundir com `docs/comercial/` (vendas).
5. Atualize links em README ou índice em `docs/README.md` se existir.
